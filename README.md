# 🌌 AstroAdventure

AstroAdventure is a web application built using **ASP.NET MVC (Framework)** that allows users to explore various astronomy-related topics such as planets, stars, and missions.  
It also includes **user authentication (login/signup)**, profile management, and an admin panel for managing content.

---

## 🚀 Features

- 🪐 Browse astronomy topics (Articles, Planets, Stars, Missions, Categories)
- 👥 User authentication (Login, Signup, Logout)
- ⚙️ Admin panel for managing users, images, and comments
- 🖼️ Profile pictures and user settings
- 📱 Responsive UI with dark theme
- 🧩 CRUD operations on data
- 📅 Display of created date and time for users

---

## 🧠 Tech Stack

| Technology | Description |
|-------------|-------------|
| **ASP.NET MVC 5** | Web application framework |
| **C#** | Backend programming language |
| **Entity Framework** | ORM for database operations |
| **SQL Server** | Database |
| **HTML5 / CSS3 / Bootstrap** | Frontend and styling |
| **JavaScript / jQuery** | Dynamic UI features |
| **Visual Studio 2022** | IDE used for development |
